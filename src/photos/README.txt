📸  ISHA'S PHOTOS — drop them right here
==========================================

Put Isha's photos in THIS folder and they will automatically appear
in the "Photo Wishes for Isha" animation on the website. No coding needed.

HOW
---
1. Copy your image files into this folder (src/photos/).
2. Name them so they sort in the order you want them to show, e.g.
       01.jpg
       02.jpg
       03.jpg
   (or descriptive: 01-paris.jpg, 02-beach.jpg, 03-cafe.jpg)
3. Save. If the dev server is running it refreshes on its own.

SUPPORTED FILE TYPES
--------------------
.jpg  .jpeg  .png  .webp  .gif  .avif

TIPS
----
• Portrait (tall) photos look best — the frame is shaped 4:5.
• Each photo is paired with a birthday wish, in order. You can edit the
  wishes in  src/app/wishes.ts  (add as many as you like).
• Add as many photos as you want — the animation loops through them all,
  one beautiful pop-up at a time.

Until you add photos here, the animation shows placeholder images so you
can preview how it looks.
