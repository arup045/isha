🎵  ISHA'S BIRTHDAY MUSIC — drop your song files right here
=============================================================

Put your audio files in THIS folder (src/music/) and they will play
automatically in the background of the website — one song after another,
on a loop. No coding needed.

HOW
---
1. Copy your song files into this folder (src/music/).
2. Name them so they sort in the order you want them to play, e.g.
       01.mp3
       02.mp3
3. Save. If the dev server is running it refreshes on its own.
4. On the website, music starts on the first tap/click anywhere
   (browsers block auto-playing sound until you interact once).
   The 🌹 "Birthday Melody" button toggles play/pause.

SUPPORTED FILE TYPES
--------------------
.mp3  .m4a  .ogg  .wav  .aac    (.mp3 is the safest choice)

NOTES
-----
• Add as many songs as you like — they all play in a loop, in order.
• These play the FULL songs for everyone (no Spotify / Premium needed).
• If this folder is empty, the site falls back to the two Spotify tracks.
